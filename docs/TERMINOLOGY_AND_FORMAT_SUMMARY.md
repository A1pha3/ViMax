### ✅ 5.2 统一文档格式

**成果**：
- 创建了 `docs/.format_guide.md` 格式规范文档
- 创建了 `scripts/check_doc_format.py` 格式检查脚本
- 定义了统一的文档结构模板
- 规范了标题层级、代码块、列表、表格等格式

**格式规范内容**：
- 文件命名规范（小写字母+下划线）
- 标题层级规范（一级标题仅一个）
- 代码块格式（必须有语言标识符）
- 列表格式（使用 `-` 而非 `*`）
- 链接格式（内部链接、外部链接）
- 术语使用规范
- 标点符号规范

**格式检查结果**：
- 检查了 13 个文档文件

- 主要问题：多个一级标题、代码块缺少语言标识符
- 提供了详细的问题报告

## 使用指南

### 如何使用术语表

1. **查找术语**：
   - 打开 `docs/glossary.md`
   - 使用目录快速定位类别
   - 查看术语的中英文对照和定义

2. **编写文档时**：
   - 参考术语表确保翻译一致
   - 首次使用术语时可添加英文原文
   - 专有名词保持英文

3. **贡献术语**：
   - 发现缺失术语时提交 Issue
   - 提供术语的定义和使用示例
   - 遵循术语表的格式规范

### 如何使用格式规范

1. **编写新文档**：
   - 参考 `docs/.format_guide.md`
   - 使用标准文档模板
   - 遵循格式规范

2. **检查文档格式**：
   ```bash
   python scripts/check_doc_format.py
   ```

3. **修复格式问题**：
   - 根据检查报告逐项修复
   - 重新运行检查确认修复
   - 提交前确保无格式问题
