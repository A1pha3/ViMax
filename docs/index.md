# ViMax 文档

欢迎来到 ViMax 文档中心。ViMax 是一个多智能体视频生成框架，旨在将您的创意、剧本或小说转化为高质量的视频内容。

## 项目简介

ViMax 通过自动化从叙事输入到最终视频输出的完整流程，解决了传统视频制作中的诸多瓶颈：

- **创意到视频 (Idea2Video)**: 从一句话创意生成完整视频。
- **小说到视频 (Novel2Video)**: 智能改编长篇小说为分集视频。
- **剧本到视频 (Script2Video)**: 将专业剧本转化为视觉叙事。
- **智能客串 (AutoCameo)**: 将用户形象融入视频故事中。
