## ✨ 核心特性

### 🎬 多种视频生成模式

ViMax 提供三种主要的视频生成流水线，满足不同的创作需求：

#### Idea2Video - 创意到视频
从一句话创意自动生成完整视频，适合快速原型制作和创意验证。

**特点**：
- 自动剧本创作和分镜设计
- 智能角色设定和场景规划
- 端到端的自动化流程

**适用场景**：短视频创作、广告制作、创意展示

#### Script2Video - 剧本到视频
将专业剧本转化为视觉叙事，保持原作的艺术表达。

**特点**：
- 精确的剧本解析和场景提取
- 专业的分镜设计和镜头规划
- 支持复杂的叙事结构

**适用场景**：影视制作、教育视频、故事讲述

#### Novel2Video - 小说到视频
智能改编长篇小说为分集视频，自动处理复杂的叙事结构。

**特点**：
- 智能内容压缩和情节提取
- 自动分集和节奏控制
- 角色一致性管理

**适用场景**：小说改编、长篇内容制作、系列视频

### 🤖 多智能体协作架构

ViMax 采用专业分工的多智能体系统，每个智能体负责特定的创作任务：

- **编剧 (Screenwriter)** - 创作和优化剧本内容
- **分镜师 (Storyboard Artist)** - 设计视觉叙事和镜头语言
- **角色设计师 (Character Designer)** - 提取和管理角色信息
- **场景规划师 (Scene Planner)** - 规划场景和环境设定
- **导演 (Director)** - 生成详细的镜头描述
- **摄影师 (Cinematographer)** - 控制镜头参数和视觉风格

详见 [智能体详解](./agents.md) 和 [系统架构](./architecture.md)

### 🔧 灵活的工具集成

ViMax 支持多种主流的图像和视频生成工具，您可以根据需求灵活切换：

**图像生成工具**：
- Doubao SeeDream (字节跳动)
- Nanobanana (Google)
- 其他自定义工具

**视频生成工具**：
- Google Veo
- Doubao SeeDance (字节跳动)
- 其他自定义工具

详见 [工具与集成](./tools.md)

### 🎨 智能角色一致性

通过角色画像 (Character Portrait) 技术，确保同一角色在不同镜头中保持视觉一致性，提升视频的专业度和连贯性。
